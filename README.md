# Sistema de Gestión de Libros Electrónicos

## Descripción

Este repositorio contiene la planeación de un sistema de gestión de libros electrónicos. El sistema permitirá administrar usuarios, libros, categorías, búsquedas, favoritos y accesos digitales.

## Objetivo

Diseñar y desarrollar un sistema que permita registrar, organizar, consultar y administrar libros electrónicos mediante principios de programación funcional.

## Etapa actual

El proyecto se encuentra en la etapa de planeación del software. En esta primera entrega no se incluye código fuente.

## Módulos previstos

- Autenticación.
- Gestión de usuarios.
- Gestión de libros.
- Gestión de categorías.
- Búsqueda.
- Favoritos.
- Préstamos o accesos digitales.

## Alcance

El sistema permitirá registrar usuarios y libros, organizar libros por categorías, realizar búsquedas, gestionar favoritos y registrar el acceso a los libros.

## Estructura del repositorio

- `documentacion`: documentos relacionados con el proyecto.
- `diagramas`: diagramas de la estructura del sistema.
- `presentacion`: material utilizado para el video explicativo.
